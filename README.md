# 🐦 Angry Birds Clone (C++, OOPs &  SFML)

A robust, 2D physics-based game inspired by Angry Birds, built entirely in C++ utilizing the Simple and Fast Multimedia Library (SFML). 

This project demonstrates a strong application of **Object-Oriented Programming (OOP)** principles to create a highly maintainable, modular codebase. It serves as a practical implementation of core game development concepts, including custom physics modeling, state machine management, and structured entity architecture.

## ✨ Features

* **Custom Physics Mechanics:** Implements core rigid-body mechanics, including accurate gravity simulation and collision restitution (bouncing).
* **Game State Management:** Utilizes a state machine approach to seamlessly handle transitions across multiple screens:
  * Main Menu
  * Active Gameplay
  * Win / Loss Screens
  * Level Transitions
* **Level Progression:** Dynamic loading and handling of multiple stages.
* **Scoring System:** Real-time tracking of player performance and target destruction.
* **Input Handling:** Smooth, intuitive mouse drag-and-release mechanics coupled with keyboard controls.

## 🎮 How to Play

You do not need to compile the code to play the game.
1. Navigate to the **[Releases](../../releases)** section on the right side of this repository.
2. Download the `SFML.-.Angry.Birds.3.zip` file from the latest release.
3. Extract the ZIP file to a local folder on your machine.
4. Run the `.exe` file to start playing!

## 🛠️ Built With

* **C++** - Core application logic and OOP architecture.
* **[SFML](https://www.sfml-dev.org/)** - (Simple and Fast Multimedia Library) used for window creation, rendering graphics, and capturing user input.
